# 📊 Employment Productivity Dashboard

![Banner](assets/banner.png)

## 📌 Overview

The Employment Productivity Dashboard is a C++ console application developed using Object-Oriented Programming principles. It helps manage employee information and evaluates employee performance through productivity metrics.

The project demonstrates the implementation of classes, constructors, functions, encapsulation, and basic business logic in C++.

---

## ✨ Features

- Add employee details
- Display employee information
- Update employee records
- Calculate employee productivity score
- Calculate employee bonus
- Track attendance
- Track working hours
- Salary management
- Department management

## 📊 Employee Information

The dashboard stores the following information:

- Employee ID
- Employee Name
- Department
- Tasks Completed
- Hours Worked
- Attendance
- Salary
- Productivity Score

---

## 🧮 Productivity Calculation

The productivity score is calculated using:

- Tasks completed
- Working hours
- Attendance percentage

---

## 💰 Bonus Calculation

Bonus is calculated according to the employee's productivity score.

Higher productivity results in higher bonuses.
